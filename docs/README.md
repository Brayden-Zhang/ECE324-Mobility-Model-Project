# Docs

This directory contains the paper source and the small set of supporting notes needed
to reproduce or understand the repository:

- `paper.tex` and `paper.pdf`: the manuscript.
- `figures/`: figures included by the paper.
- `foundation_evals.md`: command reference for the evaluation suite.
- `neurips_experiments.md`: experiment checklist for the paper campaign.
- `baselines.md`: notes on comparison baselines.

Temporary LaTeX build products are ignored by Git and can be removed with `make clean`.
